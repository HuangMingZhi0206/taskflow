## ğŸ†˜ Troubleshooting

### Settings Not Saving:

**Problem**: Settings reset after closing app  
**Solution**: 
1. Check app permissions
2. Ensure storage permission granted
3. Clear app cache if issue persists

### Accent Color Not Applying:

**Problem**: Color doesn't change everywhere  
**Solution**:
1. Completely close and restart app
2. Color applies to primary elements only
3. Some sections use fixed colors by design

### Notifications Not Working:

**Problem**: Not receiving notifications  
**Solution**:
1. Check "Enable Notifications" is ON
2. Verify specific notification types enabled
3. Check device notification permissions
4. Ensure battery optimization not blocking app

### Theme Issues:

**Problem**: Dark mode looks wrong  
**Solution**:
1. Try toggling theme mode
2. Check if system default interfering
3. Reset to defaults and reapply
